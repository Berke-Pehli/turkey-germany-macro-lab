# Turkey-Germany Macro Lab

A reproducible macroeconomics analytics and forecasting project comparing Türkiye, Germany, and the euro area using official public data sources, PostgreSQL, Python, and Power BI.

## Project Status

Project initialization and architecture setup in progress.

## Project Goal

The goal of this project is to build a refreshable macroeconomic analysis and forecasting workflow that:

- ingests official public macro data through APIs
- stores normalized time series in PostgreSQL
- creates SQL and Python analysis layers
- builds regression, classification, and time-series forecasting models
- exports reporting-ready datasets for Power BI
- presents findings through documentation, a case-study paper, a presentation, and a website layer
